# Journey

A modern blog platform built with Ruby on Rails, featuring a clean and responsive design.

## Features

- User authentication
- Blog post creation and management
- Rich text editor
- Image uploads
- Responsive design
- Search functionality
- User profiles

## Technology Stack

- Ruby 3.x
- Rails 7.x
- PostgreSQL
- Sass/SCSS
- Stimulus.js
- Turbo
- Active Storage
- Action Text

## Getting Started

### Prerequisites

- Ruby 3.x
- PostgreSQL
- Node.js
- Yarn
